# VANTA - Void Themed Clothing Brand Website

Welcome to **VANTA**, an elegant and modern e-commerce website for a void-themed clothing brand. This website features a stunning purple and white color scheme with cosmic aesthetics.

## 🌌 Features

- **Modern Design**: Sleek, minimalist interface with a cosmic void theme
- **Responsive Layout**: Fully responsive design that works on all devices
- **Interactive Elements**: 
  - Smooth scroll navigation
  - Animated floating particles
  - Mouse trail effects
  - Hover animations on products
  - Form validation and feedback
- **Product Showcase**: Grid-based product display with badges and pricing
- **Color Palette**:
  - Primary Purple: `#6d28d9`
  - Dark Purple: `#2d1b69`
  - Light Purple: `#a78bfa`
  - White: `#ffffff`
  - Off-white: `#f8f7ff`
  - Dark Background: `#0a0015`

## 🎨 Customization

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-purple: #6d28d9;
    --dark-purple: #2d1b69;
    --light-purple: #a78bfa;
    --white: #ffffff;
    /* ... etc ... */
}
```

### Modify Products
Edit the product grid in `index.html`:
```html
<div class="product-card">
    <div class="product-image purple-gradient">
        <span class="product-badge">New</span>
    </div>
    <h3 class="product-name">Product Name</h3>
    <p class="product-price">$99.99</p>
    <button class="add-to-cart">Add to Cart</button>
</div>
```

### Adjust Animations
- Modify animation speeds in `styles.css`
- Customize particle effects in `script.js`
- Adjust hover effects and transitions
